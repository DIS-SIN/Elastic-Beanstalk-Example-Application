All API end points are format as such: /api/v1/{API_CALL}/?QS_Parameter=value where v1 is the API version number.

get-courses

Search for courses.

Query string parameters

Parameter Meaning Required? Notes
&lang=en/fr Results language No Determines whether the response course titles are in English or French. Defaults to returning both.
&title-text=text Selects specific title words to search for No Limits the response to coures with text contained within titles.
&descriptions=false/true If true, returns course descriptions along with course results No Defaults to false
&course-code=text If provided, returns information for a sepcific course with course-code value No Example format is typically: A303
